## Keywords

| Keyword  | Description                                                                                                                                                                                                                                                                                                                                                                                                |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| false    | Boolean false value                                                                                                                                                                                                                                                                                                                                                                                        |
| true     | Boolean true value                                                                                                                                                                                                                                                                                                                                                                                         |
| null     | Represents a null reference                                                                                                                                                                                                                                                                                                                                                                                |
| this     | Returns a reference to the current execution context with the following properties:  Rows: any[] (JavaScript array)                                                                                                                                                                                                                                                                                        |
