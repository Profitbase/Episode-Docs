# Connecting to Finago Office / 24SevenOffice

When adding a FinagoOffice / 24SevenOffice connection, select an **existing connection** or create a new one.

<br/>

## Connection properties

| Name              | Description    |
|-------------------|----------------|
| Connection Name   | Name of the connection object.                        |
| Client ID         | A unique identifier for the client application (also called Application ID). |
| Client secret     | A confidential key used to authenticate the client.   |
| Organization ID   | The Organization ID for the customer in 24SevenOffice |


<br/>

## Creating ClientID with Secret

You can follow the instructions as documented by [Finago Office/24SevenOffice here](https://rest-api.developer.24sevenoffice.com/doc/v1/topic/topic-get-access-token).

<br/>

> [!NOTE]
> A [Dynamic Connection](./create-connection.md) can replace the default connection described here.
