# Connecting to Finago Office 

When adding a Finago Office (24SevenOffice) connection, select an **existing connection** or create a new one.

<br/>

## Connection properties

| Name              | Description    |
|-------------------|----------------|
| Name              | Name of the connection object.                        |
| Client ID         | A unique identifier for the client application (also called Application ID). |
| Client secret     | A confidential key used to authenticate the client.   |
| Organization ID   | The Organization ID for the customer in Finago Office |


<br/>

## Creating ClientID with Secret

You can follow the instructions as [documented by Finago Office here](https://rest-api.developer.24sevenoffice.com/doc/v1/topic/topic-get-access-token).

<br/>

> [!NOTE]
> A [Dynamic Connection](./create-connection.md) can replace the default connection described here.
