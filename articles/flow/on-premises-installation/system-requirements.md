
#### Application server requirements

This depends on your expected workload. The minimum recommended specs are:

-	Windows Server 2022 or higher
-	.NET 8 or higher
-	IIS Installed
-	Memory: 16 GB RAM
-	CPU: 2 cores, 2.4 GHz
-	Disk: 500 MB 



#### Database server requirements

-	SQL Server 2022 or higher
-	Memory: 8GB RAM
-	CPU: 1 core, 2.4 GHz
-	Disk: 200 MB
