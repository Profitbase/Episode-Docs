
# User chat

The **User Chat** component provides real-time, context-aware communication directly within Workbooks. It enables end users to collaborate with messages related to the specific analytical context they are viewing â€” such as a page, dashboard, report, or even a selected record or data point.

![img](../../../../../../images/changelogs/inVision25_6_1.png)

<br/>

## Properties

| Name                | Description                   |
|---------------------|-------------------------------|
| Context Mapping     | Context Mapping allows you to associate user chat [context items](context-item.md) with specific values or dynamic expressions, for example like **_department_**. [More](./howto/context-mapping.md) |
| Context text        | A text that describes what the chat is about. This text will appear in messages on the [App Header Notifications](../../components/uielements/app-header-notifications.md#context-text-and-context-link). [More](howto/context-text.md) |
| Context link        | Link to the location where the chat is located (e.g. workbook, page, with or without filter). [More](howto/context-link.md). |
